LinKapp


Linkapp es nuestro sistema de gestión empresarial integrado y modular. Acualmente esta conformado por cinco módulos: el ERP (Sistema de Planificación de Recursos Empresariales), el PMS (Sistema de Gestión de Proyectos), el CRM (Sistema de Gestión de la Relación con el Cliente), el WMS (Sistema de Gestión de Almacenes) y el FRM (Sistema de Gestión Financiera). Además estamos planificando nuevos módulos para amplear la funcionalidad del sistema. Al ser un sistema modular, se puede implementar cada módulo por separado e integrarlos a otras aplicaciónes por medio del API RestFul de LinkApp.
